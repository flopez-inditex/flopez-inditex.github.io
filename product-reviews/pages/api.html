Endpoints

  • POST /reviews — Create review
  • GET /reviews/{id} — Get review details
  • GET /products/{productId}/reviews — List product reviews
  • PATCH /reviews/{id}/status — Update moderation status

Example Payload

{
  "productId": "SKU-123",
  "rating": 5,
  "title": "Great quality",
  "comment": "Fast delivery and excellent fit",
  "author": "anonymous"
}
